# Quantlet

**Quantlet** is a focused mental math trainer for fast arithmetic, quick calculation, and numerical fluency.

It is inspired by the clean practice flow of typing trainers, but built for arithmetic. Configure your math pool, start a session, and enter a full-focus mode where every problem trains speed, accuracy, and confidence with numbers.

## Why Quantlet exists

Most mental math tools feel either too childish or too cluttered. Quantlet is built for people who want a sharper, cleaner, more focused way to practice calculation.

The goal is simple:

> Build faster arithmetic instincts through repeated, focused, high-quality practice.

Quantlet is useful for:

- mental math practice
- quick calculation training
- arithmetic speed improvement
- numerical fluency
- quant interview preparation
- trading and finance calculation practice
- students who want to get faster with numbers

## Features

- Clean welcome page before each session
- Full-focus practice mode after pressing Start
- Multiple arithmetic pools
- Addition, subtraction, multiplication, division, and mixed practice
- Infinite mode for uninterrupted training
- Mobile-friendly layout
- Option to submit answers with Enter
- Option to auto-submit when the answer is correct
- Minimal interface designed to reduce distractions
- Fast feedback loop for building calculation speed

## Practice philosophy

Quantlet is built around one idea:

> You improve faster when the interface disappears and the math stays in front of you.

Instead of showing too many settings during practice, Quantlet separates the experience into two stages:

1. **Configuration stage**  
   Choose the type of arithmetic you want to practice.

2. **Focus stage**  
   Press Start and enter a clean, distraction-free session.

This makes practice feel more intentional and helps users stay locked in.

## Use cases

Quantlet can be used for:

- warming up before math-heavy work
- improving mental arithmetic
- preparing for quantitative interviews
- building confidence with numbers
- practicing quick calculations for school, finance, trading, or engineering
- replacing random arithmetic drills with a cleaner focused system
